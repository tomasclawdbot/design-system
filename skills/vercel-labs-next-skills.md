# vercel-labs/next-skills

Next.js App Router best practices, version upgrades, and performance features like cache components.

## Install

```bash
npx skills add vercel-labs/next-skills
```

## Core Principles

- **App Router First** — Use Next.js 15 App Router patterns
- **Performance** — Cache components, optimize loading
- **Version Awareness** — Keep up with Next.js upgrades

## How to Apply

- Use Server Components by default
- Implement proper caching strategies
- Keep dependencies updated
- Follow Vercel's performance recommendations
