# Contributing

This project follows raphaelsalaja's open knowledge principle: **document everything, share freely**.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b my-feature`
3. **Make changes** following the skills in `/skills/`
4. **Document** your changes
5. **Submit a PR**

## Principles to Follow

- **Open Knowledge**: Document everything, no paywalls
- **Form + Function**: Balance beauty with utility
- **Micro-Details Matter**: Pay attention to small interactions
- **Living Documentation**: Keep docs updated with code

## Getting Started

```bash
npm install
npm run dev
```

## Adding Skills

Add markdown files to `/skills/` following the pattern:
- `{skill-name}.md`
- Include core principles, how to apply, and source links
